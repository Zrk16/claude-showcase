01The Model

It reads the room before it speaks.

Claude is built to weigh the question, not just answer it. Long context, careful reasoning, and the kind of restraint that makes a smart colleague worth keeping around.

01 / 04Reasoning

Thinks before answering.

Extended thinking turns a prompt into a private notebook of working steps, then returns the cleaner version aloud.

02 / 04Memory

Holds the whole book.

Two hundred thousand tokens of context. A novella's worth of code, papers, and conversation kept in mind at once.

03 / 04Vision

Reads the diagram.

Whiteboard photos, dashboards, ambiguous screenshots. It looks first, then talks about what it actually saw.

04 / 04Tools

Reaches for the right tool.

Calls functions, runs code, browses, files tickets. Picks the smallest tool that finishes the job.

02Claude Code

A coworker that lives in your terminal.

Claude Code is the agentic CLI. It reads your repo, plans the work, edits files, runs tests, explains itself. Not autocomplete. A pair.

~/projects/field-guide • claude 
$ claude "refactor the auth middleware to use the new session store"
● Reading src/middleware/auth.ts and three files it touches…
● The new store is in src/lib/session.ts. I'll thread it through.
● Editing src/middleware/auth.ts  +34  −18
● Editing src/middleware/__tests__/auth.test.ts  +12  −3
● Running pnpm test auth
  ✓ 14 passed in 0.42s
● Done. The middleware now reads from SessionStore; cookie names unchanged for backwards compatibility.
$ ▍
i.

Reads the repo, not the snippet.

Grep, glob, follow imports. It builds a real model of your code before it changes a line.

ii.

Plans, then commits.

Writes a step list you can edit, runs the steps, opens the diff. You stay in the loop without standing in the way.

iii.

Tests its own work.

Runs the suite, reads the failure, fixes the cause. If it can't, it says so plainly instead of pretending.

03Claude Design

Made the way a good book is made.

The visual language is not chrome. It's craft: warm paper, a careful serif, ink that rests where it falls. The interface is the manners.

A. Typography
Aa
Fraunces Display · Variable Soft 100 · Wonk on

One serif, set with intention. Body in IBM Plex. Mono in JetBrains. No system fallbacks. No surprises.

B. Palette
Paperoklch .96
Inkoklch .18
Copperoklch .62 .16
Rustoklch .48 .12

Tinted neutrals on warm paper. One earned accent in copper. Nothing screams. Nothing apologises either.

C. Motion

Spring physics, never linear. Things settle the way real things settle. Motion is a manner, not a flourish.

D. Composition

A 12-column grid that knows when to break itself. Asymmetry where the eye needs rest, alignment where it needs trust.
